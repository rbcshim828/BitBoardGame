## BitBoardGame
보드게임카페 홈페이지입니다. 

## 개발환경
### Front-end
* HTML5, CSS3
* Javascript, JQuery, Ajax
* KaKao Map API, Naver smtp

### Back-end
* JDK1.8.0
* Jackson 2.10.1
* Apache Tomcat 8.5
